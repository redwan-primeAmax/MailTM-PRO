# Redwan Mail•TM 📧

**Redwan Mail•TM** is a high-performance, privacy-focused disposable email service built with modern web technologies. Generate unique, temporary mailboxes instantly to protect your real identity and keep your primary inbox clutter-free.

---

## 🚀 Features

- **Instant Generation**: Create disposable mailboxes in a single click using premium UI caching.
- **Privacy First**: No tracking, no logs, and non-persistent local data storage.
- **Operational HUD**: Real-time metrics for engine cores, routing status, and security layers.
- **Copy-to-Clipboard**: Enhanced accessibility with one-tap address copying directly from the header.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile environments.
- **Secure Password Mgmt**: Unique custom passwords generated for every session.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **API**: [Mail.tm](https://mail.tm/) (Programmatic Gateway)

## 📦 Getting Started

### Prerequisites

- Node.js 24+
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/redwanislam7900/MailTM-PRO.git
   cd MailTM-PRO
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
